# Local AI Helper User Guide

This guide provides a list of natural language commands that you can use with the Local AI Helper.

## General Commands

- **`help`** or **`what can you do`**
  - Displays a list of available capabilities.
  - *Examples:* "help", "what can you do"

## Time and Date Commands

- **`what time is it`**, **`current time`**, or **`tell me the time`**
  - Displays the current time.
  - *Examples:* "what time is it?", "tell me the time"

- **`what's the date today`**, **`current date`**, or **`what date is it`**
  - Displays the current date.
  - *Examples:* "what's the date today?", "current date"

## File System Commands

- **`list files`** or **`show files`**
  - Lists the files in the current directory.
  - *Examples:* "list files", "show me the files"

- **`create folder <folder_name>`** or **`make folder <folder_name>`**
  - Creates a new folder with the specified name.
  - *Examples:* "create folder my_new_folder", "make a folder called images"

- **`open folder`** or **`open current folder`**
  - Opens the current directory in the file explorer.
  - *Examples:* "open folder", "open the current folder"

- **`delete file <filename>`**, **`remove file <filename>`**, or **`del <filename>`**
  - Deletes a file.
  - *Examples:* "delete file old_file.txt", "remove notes.md"

## Web Commands

- **`open browser`**
  - Opens the default web browser to Google.
  - *Examples:* "open browser", "launch chrome"

- **`search for <query>`** or **`search <query>`**
  - Searches the web for the given query.
  - *Examples:* "search for python tutorials", "search how to make a cake"

## System Commands

- **`system info`** or **`system information`**
  - Shows basic system information.
  - *Examples:* "system info", "show system information"

- **`my ip`** or **`ip address`**
  - Shows the network IP configuration.
  - *Examples:* "my ip", "what is my ip address"

- **`disk usage`** or **`free space`**
  - Shows disk usage information.
  - *Examples:* "disk usage", "show free space"

- **`show processes`** or **`list processes`**
  - Shows a list of running processes.
  - *Examples:* "show processes", "list running tasks"

## Other Commands

- **`weather in <location>`**
  - Gets the weather forecast for the specified location.
  - *Examples:* "weather in London", "what's the weather in New York"

- **`run <command>`** or **`execute <command>`**
  - Executes a custom shell command.
  - *Examples:* "run git status", "execute python --version"
