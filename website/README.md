# Nexus AI Website

A professional Streamlit website showcasing AI services and the DeepSync desktop app.

## Quick Start

```bash
cd website
pip install -r requirements.txt
streamlit run app.py
```

Or on Windows, double-click `run_website.bat`.

## Features

- **Home** – Hero section, stats, featured product, services grid
- **Local Agent** – Dedicated page with features, quick start, download
- **Products** – Product catalog with download buttons
- **Contact** – Contact form
- **FAQ** – Common questions and answers

## Download

The website includes a one-click download of the DeepSync desktop app (zip with source code).
